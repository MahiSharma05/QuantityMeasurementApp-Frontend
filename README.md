# 📏 Quantity Measurement App - Frontend

A modern frontend application for performing **unit and quantity conversions** such as length, weight, temperature,Volume.
This project includes two implementations:

* 🌐 HTML, CSS, JavaScript
* ⚡ **Angular**

---

## 🚀 Features

*  Convert between different measurement units
*  Clean and user-friendly interface
*  Fast and responsive UI
*  Integration with backend microservices
*  Authentication support (Angular version)
*  Multiple frontend implementations
